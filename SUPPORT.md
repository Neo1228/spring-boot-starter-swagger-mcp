# Support

Thank you for using `spring-boot-starter-swagger-mcp`. We want to help you get the most out of this project.

## How to get help

If you have a question or need help, please follow these steps:

1.  **Search the documentation:** Check the `README.md` and other documentation files in the repository.
2.  **Search existing Issues:** Your question may have already been asked and answered. Check both [open and closed issues](https://github.com/Neo1228/spring-boot-starter-swagger-mcp/issues).
3.  **Open a New Issue:** If you can't find an answer, please [open a new issue](https://github.com/Neo1228/spring-boot-starter-swagger-mcp/issues/new).

## What to include in your request

When asking for help, please provide as much information as possible, including:

*   **Version:** Which version of the starter are you using?
*   **Environment:** Spring Boot version, Java version.
*   **Description:** A clear and concise description of the problem or question.
*   **Steps to Reproduce:** (If applicable) How can we reproduce the issue?
*   **Logs/Errors:** Any relevant log output or error messages.

## Response time

This is an open-source project maintained by volunteers. We will try to respond as soon as possible, but please be patient.
