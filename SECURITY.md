# Security Policy

## Supported Versions

Security fixes are provided for the latest released minor line.

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1.0 | No        |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use one of these paths:

1. GitHub Security Advisory (preferred)
2. Private maintainer contact through repository owner profile

Include:

- Affected version
- Reproduction steps
- Expected vs actual behavior
- Suggested mitigation (if known)

You can expect:

- Initial response within 7 days
- Confirmation once reproduced
- A coordinated fix and release note after patching
